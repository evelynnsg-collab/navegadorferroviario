# Navegador Ferroviario SP

Aplicativo de rotas para a rede ferroviaria de Sao Paulo - Metro + CPTM.

Funciona offline como PWA.

## Linhas cobertas
1-Azul, 2-Verde, 3-Vermelha, 4-Amarela, 5-Lilas, 7-Rubi, 8-Diamante, 9-Esmeralda, 10-Turquesa, 11-Coral, 12-Safira, 13-Jade, 15-Prata e 17-Ouro.

Projeto criado e desenvolvido por Evelyn Santos de Almeida Nunes
